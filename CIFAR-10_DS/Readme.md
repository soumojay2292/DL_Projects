# DenseNet on CIFAR-10

This project implements DenseNet from scratch using PyTorch on CIFAR-10 dataset.

## Features
- Dense connectivity
- Dense blocks and transition layers
- Training with tqdm progress bar
- Evaluation with accuracy and confusion matrix
- Visualization using matplotlib

## Results
Achieved ~75–85% accuracy depending on training.

## How to Run
1. Open the notebook in Google Colab
2. Run all cells
3. View outputs and graphs

## Tech Stack
- PyTorch
- Python
- Matplotlib
- TQDM
